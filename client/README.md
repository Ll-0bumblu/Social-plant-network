# 🌿 Plant Social

Веб-приложение для растений с умным детектором и календарём ухода.
Учебный проект. Прототип приложения для демонстрации концепции.

## 🚀 Как это работает
Проект объединяет три основных экрана в одном интерфейсе:
__1️⃣ Мой сад__
![Картинка](./images/garden-page.png)
- Список всех растений пользователя
- Кнопка "Добавить растение"

__2️⃣ Детектор растений__
![Картинка](./images/detector-page.png)
- Загрузка фото растения
- Распознавание через Plant.id API
- Отображение точности совпадения и добавление растения в сад

__3️⃣ Календарь ухода__
![Картинка](./images/garden-page.png)
- Календарь ухода
- Список ближайших событий по уходу за растениями
